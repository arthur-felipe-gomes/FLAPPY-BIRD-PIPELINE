  Flappy Bird em Python com Pipeline

Este projeto é um jogo inspirado no clássico Flappy Bird, desenvolvido em Python utilizando a biblioteca pygame.

O objetivo do jogo é controlar o personagem e desviar dos obstáculos, acumulando pontos a cada passagem bem-sucedida.

---

##  Funcionalidades

- Sistema de pontuação
- Sistema de vida
- Sons para ações do jogo (passar pelos canos e morte)
- Tela de fim de jogo com opção de reiniciar
- Interface gráfica simples e interativa

---

##  Tecnologias Usada

- Python
- Pygame
- PyInstaller (para geração do executável)

---

##  Pipeline

O projeto utiliza pipeline para automatizar processos importantes do desenvolvimento, como:

- Execução automática do código
- Validação de funcionamento
- Preparação para entrega (build do executável)

A pipeline ajuda a garantir que o jogo esteja sempre funcionando corretamente antes de ser compartilhado ou atualizado.

---

##  Como executar o projeto

### Rodando com Python:
1. Instale o Python
2. Instale o pygame:
   pip install pygame
3. Execute o arquivo principal:
   python main.py

---

### Rodando sem instalar Python:
1. Baixe o arquivo compactado (.zip)
2. Extraia os arquivos
3. Execute o arquivo main.exe

---

##  Estrutura do projeto

- main.py → arquivo principal do jogo
- assets → imagens e sons
- dist → versão executável do jogo
